# [BETA] 신호등 알리미 Pro (Traffic Light Pro)

> **주의: 본 서비스는 현재 v0.4 테스트 및 개발 단계에 있습니다.**
> 본 소프트웨어는 저시력 보행자를 위한 '보조 도구'일 뿐이며, 신호 판단의 최종 책임은 사용자에게 있습니다. 
> 야간, 역광, 우천 시 AI 인식률이 저하될 수 있으므로 전적으로 의존하지 마십시오.

---

## 1. 프로젝트 개요
신호등 알리미 Pro는 저시력 보행자가 횡단보도 신호를 보다 안전하게 인지할 수 있도록 돕는 **실시간 AI 비전 보조 시스템**입니다. 구글 MediaPipe 엔진을 활용하여 별도의 앱 설치 없이 웹 브라우저에서 즉시 실행됩니다.

* **서비스 URL:** https://mirnet96.github.io/traffic-light
* **엔진:** Google MediaPipe (EfficientDet Lite0)
* **상태:** v0.4 Beta (2026.03 업데이트)

---

## 2. 핵심 기능 (Key Features)

### AI 객체 탐지 및 검증
- **MediaPipe EfficientDet:** 고속 프레임 분석을 통한 실시간 신호등 추적.
- **시계열 검증 시스템:** 빛 번짐이나 간판 등의 노이즈를 걸러내기 위해 연속된 프레임이 일치할 때만 안내를 시작합니다.

### 다중 피드백 시스템 (Multimodal Feedback)
- **지능형 비프음(Beep):** 초록불(고음/짧은 주기), 빨간불(저음/긴 주기)을 소리만으로 구분.
- **강력한 진동 패턴:** 신호 변화 시 모바일 기기 진동을 통한 촉각 알림.
- **TTS 음성 안내:** 현재 신호 상태 및 보행 안전 수칙 한국어 가이드.

### 스마트 가이드 및 최적화
- **기울기 감지(Gyroscope):** 기기 각도가 부적절할 경우 "폰을 세워주세요" 배너 표시.
- **자동 줌 탐색:** 신호등이 포착되지 않을 경우 카메라 배율을 자동으로 조정.
- **야간 모드:** 빛 번짐 억제를 위한 카메라 노출(Exposure) 자동 보정 엔진 통합.

## 3. 사용 방법 (How to Use)

1. **접속:** 크롬 또는 삼성 인터넷 브라우저로 서비스 URL 접속.
2. **권한 허용:** 카메라, 오디오, 자이로 센서 권한을 반드시 '허용'하십시오.
3. **시스템 가동:** 화면 중앙의 [시스템 가동] 버튼 클릭.
4. **위치 고정:** 기기를 정면으로 높게 들어 건너편 신호등을 비춥니다.
5. **안내 수신:** 음성, 진동, 비프음 주기에 따라 주변 상황을 확인하며 이동합니다.
